Desafío 3 - Funciones



Desafío Parte 1

  • 1. Transforma esta declaración de función a una función de expresión. El ejercicio debe quedar en un archivo nombrado `1funcion.js` (1 Punto)
    •             
                function example(a, b, c){
                  return a+b+c
                  }

    Respuesta:

              
                suma = function sumastres(a, b, c){
                  return a+b+c;
                  }



    Desafío Parte 2

  • 2. Transforma las siguiente función a una arrow function de una línea. Este ejercicio debe quedar en un archivo llamado `2arrow.js` (1 Punto)
    • 
          suma = function(a, b){
            return a + b
            }

    Respuesta:

    
          suma = (a, b) => a + b;
